# Brief

Vous devez intégrer le composant ProductCard avec React.
Vous partirez de la maquette HTML, créerez les fichiers nécessaires et utiliserez le HTML et les CSS pour construire un composant autonome

# ProductCard

## Props

Les props suivants doivent être intégrés :

hot: true
image: "https://i.imgur.com/xdbHo4E.png"
category: "Women,bag"
title: "WOMEN LEATHER BAG"
description: "Lorem ipsum dolor sit amet, consectetur adipisicing elit. Vero, possimus nostrum!"
price: 230.99
price_sold: 89.99

## Comportement

Si price_sold est transmis, le pris est barré et price_sold apparait.
